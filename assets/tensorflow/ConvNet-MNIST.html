ConvNet¶

Chaque couche de convolution possde des filtres de dimension [hauteur, largeur, canaux_entre, canaux_sortie]. canaux_entre vaut 1, puisque l'on travaille avec des donnes en chelles de gris. canaux_sortie vaut 10 puisqu'on a 10 valeurs diffrentes prdire en sortie du rseau.

On utilise la fonction tf.nn.conv2d pour crer des couches qui effectuent les convolutions. Cette fonction prend en entre le tenseur x et les poids W appliquer, et utilise la fonction d'activation reLu.

# Couche de convolution 2D
def conv2d(x, W, b, strides=1):
    x = tf.nn.conv2d(x, W, strides=[1,1,1,1], padding='SAME')
    x = tf.nn.bias_add(x, b)
    return tf.nn.relu(x)

On dfinit une fonction globale pour le modle conv net :

  • Elle prend en argument le dataframe, les poids, les biais.
  • Les poids et biais seront utiliss dans les diffrentes couches.
  • Stride est le pas de dcalage de la fentre de convolution.

En utilisant une fenetre de 2x2 (ksize=[1,2,2,1]) et un stride de 2 (strides=[1,2,2,1]), les dimensions vont tre rduites de moiti. Cela aide rduire la taille du modle en conservant les composantes les plus importantes.

Aprs la dernire convolution, les donnes sont reshapes avant de passer dans la couche fully-connected.

On ajoute du drop out, uniquement pour l'entrainement du modle. Drop Out : On dsactive alatoirement certain neurones, afin d'intruire une composante alatoire dans notre rseau.

Evolution des dimensions de donnes :

  • En entre, nos images sont au format 28x28.
  • Les deux couches de max pooling rduisent chacune de moiti les dimensions. Les couches orange fonc sur le graphique. On a donc (28 / 2) /2 = 7.

def conv_net(data, weights, biases, training=False):
    # Convolution layers
    conv1 = conv2d(data, weights['c1'], biases['c1']) # [28,28,16]
    conv2 = conv2d(conv1, weights['c2'], biases['c2']) # [28,28,16]
    pool1 = tf.nn.max_pool(conv2, ksize=[1,2,2,1], strides=[1,2,2,1], padding='SAME') 
    # [14,14,16]

    conv3 = conv2d(pool1, weights['c3'], biases['c3']) # [14,14,32]
    conv4 = conv2d(conv3, weights['c4'], biases['c4']) # [14,14,32]
    pool2 = tf.nn.max_pool(conv4, ksize=[1,2,2,1], strides=[1,2,2,1], padding='SAME') 
    # [7,7,32]

    # Flatten
    flat = tf.reshape(pool2, [-1, weights['d1'].get_shape().as_list()[0]]) 
    # [7*7*32] = [1568]

    # Fully connected layer
    fc1 = tf.add(tf.matmul(flat, weights['d1']), biases['d1']) # [128]
    fc1 = tf.nn.relu(fc1) # [128]

    # Dropout
    if training:
        fc1 = tf.nn.dropout(fc1, rate=0.2)

    # Output
    out = tf.add(tf.matmul(fc1, weights['out']), biases['out']) # [10]
    return out

from keras.models import Sequential, Model
from keras.layers import Dense, Activation, Dropout, Flatten
from keras.layers import Conv2D, MaxPooling2D, Input
from keras.optimizers import Adam
from keras.datasets import mnist
from keras.utils import to_categorical
import numpy as np

# MNIST Dataset
(train_images, train_labels),(test_images, test_labels) = mnist.load_data()
train_images  = np.expand_dims(train_images.astype(np.float32) / 255.0, axis=3)
test_images = np.expand_dims(test_images.astype(np.float32) / 255.0, axis=3)
train_labels = to_categorical(train_labels)
test_labels = to_categorical(test_labels)

# Training parameters
batch_size = 128
n_epochs = 5
n_classes = 10

# Create the model
model = Sequential()

# Convolution Layers
model.add(Conv2D(16, 3, activation='relu', input_shape=(28, 28, 1), padding='same'))
model.add(Conv2D(16, 3, activation='relu', padding='same'))
model.add(MaxPooling2D(pool_size=2, padding='same'))
model.add(Conv2D(32, 3, activation='relu', padding='same'))
model.add(Conv2D(32, 3, activation='relu', padding='same'))
model.add(MaxPooling2D(pool_size=2, padding='same'))

# Dense Layers
model.add(Flatten())
model.add(Dense(128, activation='relu'))
model.add(Dropout(0.2))
model.add(Dense(n_classes, activation='softmax'))

# Optimizer
optimizer = Adam(lr=1e-4)

# Compile
model.compile(loss='categorical_crossentropy',
              optimizer=optimizer,
              metrics=['categorical_accuracy'])

# Train model
model.fit(train_images, 
          train_labels,
          epochs=n_epochs,
          batch_size=batch_size,
          validation_data=(test_images, test_labels)
          )

# Show Sample Predictions
predictions = model.predict(test_images[:25])
predictions = np.argmax(predictions, axis=1)
f, axarr = plt.subplots(5, 5, figsize=(25,25))
for idx in range(25):
    axarr[int(idx/5), idx%5].imshow(np.squeeze(test_images[idx]), cmap='gray')
    axarr[int(idx/5), idx%5].set_title(str(predictions[idx]),fontsize=50)

# Save Model
model.save_weights("model.h5")
